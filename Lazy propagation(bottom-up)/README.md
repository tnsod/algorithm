재귀를 사용하지 않고 바텀업 세그먼트 트리의 lazy propagation 구현

구간 업데이트, 구간 합 구현 예시
<br>
<br>
<br>
update 함수 : 

step 1: 업데이트 하려는 구간을 포함하는 부모노드의 lazy 값을 자식에게 전파

step 2: 구간 갱신 (일반 비재귀와 동일), 부모에 lazy값 등록

step 3: Step 2에서 변경된 자식 노드들의 값을 바탕으로, 부모 노드들의 값을 다시 계산
<br>
<br>
<br>
query 함수 :

step 1: 구하려는 구간을 포함하는 부모노드의 lazy 값을 자식에게 전파

step 2: 구간 쿼리 구하기 (일반 비재귀와 동일)
